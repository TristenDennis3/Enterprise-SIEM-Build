The Network

An air-gapped model of enterprise detection

A fully segmented, air-gapped SIEM Network that simulates attacks and detects them across network and host sensors. Four isolated VLANs, dual SIEM platforms, live adversary emulation, and custom detection engineering. It also serves as a hands-on learning resource for future cybersecurity students at Northern Michigan University.

How It Works

The Detection Pipeline

Every log source feeds the SIEM core, which normalizes, enriches, correlates, and surfaces events through to analyst response.
