The Stack

Software

Every platform and tool in the network, grouped by function. Hover a row to highlight it.

Infrastructure
ToolRole in the network
pfSense+ WebGUIUsed to create firewall rules and view logs of network traffic. All traffic passes through here.
UniFi Network ControllerManages the UniFi switch ports, VLAN tagging, and the SPAN mirror port.
ProxmoxHypervisor managing all VMs: deployment, snapshots, and resource allocation.
Linux VMsHost vulnerable services and web apps (Metasploitable 3, DVWA) used as attack targets.
Active DirectoryWindows Server 2022 domain controller (siemlab.local). The target for credential attacks, privilege escalation, and lateral movement.
Detection / SIEM
ToolRole in the network
Security OnionNetwork SIEM platform capturing and analyzing mirrored SPAN traffic (Suricata + Zeek).
SuricataRule-based IDS/IPS analyzing mirrored traffic for known attack signatures and anomalies.
ZeekNetwork security monitor generating rich connection and protocol logs from mirrored traffic.
WazuhHost-based EDR/HIDS with agents on every machine that catch local execution and suspicious activity.
NessusVulnerability scanning of network and endpoints.
Attack / Tooling
ToolRole in the network
Kali LinuxOffensive host in VLAN 40. Base for penetration testing (Metasploit, Impacket, Hydra, Wireshark).
MITRE CalderaAutomated adversary emulation with scripted attack chains that play the role of a threat actor.
Sliver C2Post-exploitation command-and-control for beaconing and attacker-persistence practice.
HydraCredential brute-force testing (e.g., RDP) against victim hosts.
WiresharkPacket capture and analysis for validating traffic and confirming detections.
Metasploitable 3Intentionally vulnerable Linux host used for exploitation practice.
DVWADamn Vulnerable Web App for web-app testing: SQLi, XSS, and command injection.
