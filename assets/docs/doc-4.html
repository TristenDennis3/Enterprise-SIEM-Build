SECURITY ONION — ATTACKER & TARGET HUNTING

Query

Purpose

source.ip:10.0.40.200

All traffic originating from the attacker (Kali) — a single query to see the full campaign

source.ip:10.0.40.200 | groupby event.dataset

Attacker activity broken down by data type (alerts, conn, dns, http, etc.)

source.ip:10.0.40.200 | groupby destination.ip

Every host the attacker touched

source.ip:10.0.40.200 | groupby destination.port

The services and ports the attacker targeted

source.ip:10.0.40.200 AND event.severity_label:high

The attacker's activity that triggered high-severity alerts

destination.ip:10.0.30.10 | groupby event.dataset

Everything hitting the domain controller, by data type

destination.ip:10.0.30.40

Traffic to the Windows 11 host (lateral movement target)

destination.ip:10.0.30.80

Traffic to DVWA (web application attacks)

destination.ip:10.0.30.30

Traffic to Metasploitable (ProFTPD exploitation)

SECURITY ONION — ALERTS & ZEEK DATASETS

Query

Purpose

event.dataset:alert | groupby rule.name

Every Suricata alert type that fired, grouped by signature

event.module:zeek | groupby event.dataset

Lists all available Zeek datasets on the sensor

event.dataset:zeek.kerberos

Kerberos activity — where Kerberoasting ticket requests appear (port 88)

event.dataset:zeek.conn

Raw connection logs — source/dest, ports, bytes, connection state

event.dataset:zeek.http

Web request logs — where the DVWA attacks appear

event.dataset:zeek.dns

DNS query logs — useful for spotting tunneling or beaconing patterns

WAZUH — HOST & WINDOWS EVENT HUNTING

Query

Purpose

data.win.system.eventID:4624

Successful logons — surfaces the lateral movement authentications

data.win.system.eventID:4769

Kerberos service ticket requests

data.win.system.eventID:4672

Special privileges assigned — admin-level logon sessions

rule.mitre.id:T1550.002

Pass-the-Hash

rule.mitre.id:T1078

Valid Accounts

agent.name:DC01

All events reported by the domain controller agent

rule.level:7

Wazuh alerts at rule level 7

rule.description:*sudo*

Catches privilege-escalation activity (e.g. sudo to root on the Linux targets)

NOTES

Security Onion's Hunt interface returns far more detail than the Alerts page, since it exposes the underlying Zeek logs and not just the events that triggered a signature.
