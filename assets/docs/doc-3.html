1. Overview

I ran a non-credentialed Basic Network Scan with Nessus against the victim network. The scan covered four hosts and returned 57 vulnerabilities. Most of the findings are on Metasploitable3 (10.0.30.30), which is expected since it is built to be vulnerable. This document lists the findings, scores them with CVSS, and ranks them by risk so the most important ones get fixed first.

3. Critical Findings Detail

The three criticals are the priority. Each one allows serious access to the host.

Critical Finding

Detail

Ubuntu 14.04.x End of Life

The host runs Ubuntu 14.04.x, which is no longer maintained. It receives no security updates, so any new vulnerability stays unpatched. Upgrade to a supported release.

ProFTPD mod_copy

ProFTPD is affected by an information disclosure flaw in mod_copy. The SITE CPFR and SITE CPTO commands are available to unauthenticated clients, letting a remote attacker read and write arbitrary web-accessible files on the host.

Drupal Coder RCE

The Drupal coder module (coder_upgrade.run.php) improperly validates user input passed to an unserialized function. An unauthenticated attacker can send a crafted request to execute arbitrary PHP code, giving full remote code execution.

5. Risk Prioritization

Ranking is based mostly on CVSS, but I bumped a couple of findings using EPSS. A high EPSS means the vulnerability is likely to be exploited in the wild, so it deserves attention even if its base score is not the highest.
